# Projekt HTML/CSS – Technikum Informatyczne (2019)

## Opis projektu
Projekt został wykonany w 2019 roku w trakcie nauki w Technikum Informatycznym jako jedno z pierwszych ćwiczeń z zakresu tworzenia stron internetowych. Celem projektu było poznanie podstaw struktury dokumentu HTML oraz stylowania przy użyciu CSS.

Projekt ma charakter edukacyjny i dokumentuje moje pierwsze kroki w obszarze frontend web developmentu.

## Zakres projektu
- tworzenie struktury strony w HTML
- podstawowe stylowanie przy użyciu CSS
- praca z nagłówkami, listami, linkami i sekcjami strony
- nauka poprawnej struktury dokumentu HTML

## Technologie
- HTML5
- CSS3

## Status projektu
Projekt zakończony – repozytorium ma charakter archiwalny i służy jako dokumentacja rozwoju umiejętności na przestrzeni lat.

## Autor
Projekt wykonany samodzielnie w ramach nauki w Technikum Informatycznym.
